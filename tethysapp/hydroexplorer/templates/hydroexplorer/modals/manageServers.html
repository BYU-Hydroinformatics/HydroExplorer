HydroCatalog Settings Interface

Instructions

Add/Delete a HydroServer

You can add a SOAP endpoint to an existing HydroServer or choose from a list of HydroServers present in the catalogs


Add/Delete an CUAHSI Compatible Catalog Service

You can add an endpoint to a CUAHSI Compatible Catalog Service. This would allow you to add HydroServers from the CUAHSI Compatible Catalog Service.

Please click on the Delete button if you would like to delete an existing CUAHSI Compatible Catalog Service
